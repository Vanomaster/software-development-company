Файл с резервной копией базы данных находится в корне репозитория и называется "SDC.bak".

Название сервера можно изменить в файле по пути "Путь к директории с исполняемым файлом\Configuration\config.json" в 3 строке в пункте "Data Source="Название сервера". По умолчанию "Data Source=.\\".

Авторизоваться в программе можно, используя следующие учетные записи:
1) Учетная запись администратора: логин - 1, пароль - 1.
2) Учетная запись сотрудника: логин - 2, пароль - 2.
3) Учетная запись заказчика: логин - 3, пароль - 3.
